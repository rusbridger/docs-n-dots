# Docs 'N Dots

Some files and folders should be renamed with dots when loaded to the home directory.

- `config -> ~/.config`
- `gitconfig -> ~/.gitconfig`
- `ncmpcpp -> ~/.ncmpcpp`
- `xmonad -> ~/.xmonad`

## Scripts

Create a directory `~/scripts` and copy the following files into it.

- `audio/audio.sh`
- `brightness/brightness.sh`
